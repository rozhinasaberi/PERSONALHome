# PERSONALHome

A static personal website repository containing resume materials, academic documents, and simple HTML/CSS site assets.

## What This Project Is About

This repo appears to be an earlier personal-homepage style project focused on presenting personal documents and a static web page rather than a dynamic application.

This is a **static personal website repository**.

## Contents

- `cv/midterm.html` - main HTML page
- `cv/midterm1.css` - stylesheet
- resume and report PDFs
- image assets used by the page

## Tech Stack

- HTML
- CSS

## Open Locally

You can open the page directly in a browser or serve the `cv/` folder locally.

```bash
cd cv
python3 -m http.server 5500
```
